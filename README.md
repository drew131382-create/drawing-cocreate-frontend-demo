# 绘画共创前端 demo

这个文件夹是从 `E:\Code个人项目\绘画辅助` 中提取出来的前端页面最小可运行版本，来源主要是 `magenta-js/sketch/demos/interactive_predict`。

当前已经调整为“视频演示样机版”，不再依赖任何模型文件或 AI 推理逻辑。

## 已包含内容

- `index.html`：页面入口
- `style.css`：页面样式
- `ui-demo.js`：视频样机的前端画布交互逻辑

## 当前限制

- 当前版本不加载本地模型，也不加载远程模型
- 所有笔画都由用户手工绘制
- 按“每轮笔数”自动在人工作画（黑色）和 AI 作画（蓝色）之间轮换
- 画布左下角会显示当前轮次角色，右下角可点击“完成”
- 点击“完成”后会弹出“可以进入下一关”的卡通弹窗
- 进入下一关后会清空画布并从第 1 笔黑色人工作画重新开始
- 可以直接在画布上绘制、撤回、清空和导出

## 启动方式

推荐直接运行：

```powershell
powershell -ExecutionPolicy Bypass -File .\start-demo.ps1
```

启动后打开：

```text
http://127.0.0.1:8080
```
